# video-analyst — анализ видео и аудио

Скилл для скачивания, транскрибации и анализа видео — с YouTube и из локальных файлов.

## Источники видео

1. **Путь к файлу или папке** — пользователь даёт путь, берём файл напрямую
2. **Ссылка** — скачиваем через yt-dlp

## Триггеры

Активируй когда пользователь:
- Даёт путь к файлу или папке с видео/аудио
- Даёт ссылку на YouTube / Vimeo / другую платформу
- Говорит «посмотри видео», «разбери видео», «выжимка из видео»

## Алгоритм работы

### Шаг 1 — Получить аудио

**Если путь к файлу:**
```bash
mkdir -p /tmp/video-analyst
# видео → извлечь аудио
ffmpeg -i /ПУТЬ/К/ФАЙЛУ.mp4 -q:a 0 -map a /tmp/video-analyst/audio.mp3
# если уже аудио — просто скопировать
cp /ПУТЬ/К/ФАЙЛУ.mp3 /tmp/video-analyst/audio.mp3
```

**Если путь к папке — обработать все файлы:**
```bash
mkdir -p /tmp/video-analyst
for f in /ПУТЬ/К/ПАПКЕ/*.{mp4,mov,avi,mkv,mp3,m4a,wav}; do
  [ -f "$f" ] && ffmpeg -i "$f" -q:a 0 -map a "/tmp/video-analyst/$(basename "$f" | sed 's/\.[^.]*$//').mp3" 2>/dev/null
done
```

**Если ссылка (YouTube и др.):**
```bash
export PATH="$PATH:/home/agent/.local/bin"
mkdir -p /tmp/video-analyst
yt-dlp -x --audio-format mp3 --audio-quality 0 -o "/tmp/video-analyst/%(title)s.%(ext)s" "ССЫЛКА"
```

Если нужно само видео (для анализа визуальных элементов — графики, экран терминала и т.д.):
```bash
yt-dlp -f "best[height<=720]" -o "/tmp/video-analyst/%(title)s.%(ext)s" "ССЫЛКА"
```

### Шаг 2 — Транскрибировать

```bash
export PATH="$PATH:/home/agent/.local/bin"
whisper "/tmp/video-analyst/ФАЙЛ.mp3" --language Russian --model small --output_dir /tmp/video-analyst
```

Если язык английский: `--language English`
Если язык неизвестен: убери `--language` — whisper определит сам.

Модели по скорости/качеству: `tiny` (быстро) → `small` → `medium` → `large` (лучше всего, медленнее).

### Шаг 3 — Прочитать транскрипт

```bash
cat /tmp/video-analyst/ФАЙЛ.txt
```

### Шаг 4 — Анализ

После получения транскрипта — проанализируй содержимое под задачу пользователя:

**Для трейдинга:**
- Какую стратегию или подход описывает автор?
- Конкретные правила входа/выхода в сделку
- Индикаторы, таймфреймы, инструменты
- Управление капиталом и риск-менеджмент
- Спорные или сомнительные утверждения — отметь отдельно
- Итоговый вывод: стоит ли применять, насколько реалистично

**Для общего контента:**
- Главная идея (1-2 предложения)
- 5-7 ключевых тезисов
- Практические выводы
- Что можно применить прямо сейчас

### Шаг 5 — Очистка

```bash
rm -rf /tmp/video-analyst
```

## Форматы вывода

**Конспект:** структурированный список тезисов с таймкодами (если есть в транскрипте)

**Выжимка:** 1 страница — суть без воды

**Критический разбор:** тезисы + оценка каждого (реалистично / сомнительно / не проверено)

**Сравнение:** если дано 2+ видео — сравнить подходы, выделить общее и противоречия

## Правила

- Если видео на русском — модель `small`, быстро
- Если видео длиннее 60 минут — предупреди, транскрибация займёт 10-20 минут
- Если yt-dlp не может скачать (приватное, заблокировано) — сообщи сразу
- Всегда убирай /tmp/video-analyst после работы
- Файлы НЕ отправляй пользователю — только текстовый анализ
